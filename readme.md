# vz-train
> a small module with functions to access remote train API

## Install

Install it with

```
npm install vz-train
```
## Usage

```
Usage:
    vz-train bot
    vz-train ( -h | --help )

Options:
    -h, --help              help for vz-train
    
Commands:
    bot                     launch the bot
    
Arguments:

```
